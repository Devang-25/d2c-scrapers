  Statement 

Given N,A,B,C, find how many solutions exist to the equation : a + b + c \u2264 N, such that 0 \u2264 a \u2264 A, 0 \u2264 b \u2264 B, 0 \u2264 c \u2264 C.


Input 

The first line contains the number of test cases T. Each test case contains 4 integers, N,A,B,C. 0 \u2264 N,A,B,C \u2264 2500


Output 

Output T lines, one for each test case.


Sample Input 

2
4 3 2 1
1 1 1 1


Sample Output 

20
4
